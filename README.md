MAC CHAT
==============

* [1] 聊天（发送文本，图片，语音, 表情）（包括个人以及群组聊天）
* [2] 语音，视频通话
* [3] 创建群组 讨论组 直接输入账号发起会话
* [4] 群组 讨论组 联系人列表的显示 双击开始聊天

* [注意注意] 静态库有点大  在压缩包里 需要将静态库拖入ytx_mac_sdk文件夹下面的lib里面
